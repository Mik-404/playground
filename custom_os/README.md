# Сдача лабораторных работ

Условия лабораторных работ лежат в папке [labs/](labs/).

Для сдачи домашки:
1. Создать новую ветку (`git checkout -b <название задачи>`)
1. Отослать корректный код: задача должна пройти все тесты.
1. Открыть MR в ветку `main` с названием `HW <название задачи> <имя> <фамилия>`.
1. Дождаться проверки вашего решения.
