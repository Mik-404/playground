
# slab-allocator [200 баллов]
Как мы уже обсуждали на лекциях, HellOS очень часто выделяет много маленьких объектов (например, `mm::Task`). Такие объекты обычно довольно маленькими (несколько десятков байт), поэтому использовать по странице на каждый объект – слишком расточительно. В этом задании вам предлагается реализовать простой SLOB-аллокатор, позволяющий «нарезать» страницы, выдаваемые `mm::PageAlloc` на объекты заданного типа.
Код для задания можно найти в [`mm/obj_alloc.cpp`](../mm/obj_alloc.cpp)

Тесты для этой задачи компилируются с помощью флага `TEST_SLAB_ALLOC`.
