
# fpu-context [200 баллов]
В текущей версии HellOS отсутствует поддержка инструкций SSE/x87, что ограничивает функциональность пользовательских программ. Ваша задача — исправить эту проблему, обеспечив поддержку данных инструкций в процессоре и реализовав механизм переключения контекста для SSE/x87.

Для сохранения контекста SSE/x87 используйте инструкцию `fxsave`, а для его восстановления — `fxrstor`. Подробное описание работы этих инструкций вы найдёте в ISDM, Vol. 1, 10.5. Важно помнить, что для включения этих инструкций необходимо установить флаг `CR4.OSFXSR` в регистре `cr4`. Место хранения FPU контекста вы можете выбрать самостоятельно.

Чтобы не замедлять переключения контекста, сохраняйте и восстанавливайте состояние FPU только в случае, если процесс уже использовал инструкции SSE/x87 хотя бы один раз. По умолчанию в x86 инструкции SSE/x87 отключены, но их можно активировать, установив флаг `CR0.MP` и сняв флаг `CR0.EM` (ISDM, Vol. 3, 2.5). Для отслеживания использования процессом инструкций SSE/x87 могут быть полезны флаги `CR4.OSXMMEXCPT` и `CR0.TS`.

Тесты для этой задачи работают в user space. Чтобы их запустить, достаточно добавить флаг `TEST_FPU` при сборке через `caos-make`:
```
caos-make clean && caos-make -j... TEST_FPU=1
```
