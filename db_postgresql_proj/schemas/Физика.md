| Атрибут | Описание | Тип данных | Ограничения |
| :--- | :--- | :--- | :--- |
| 1. Developer |
| developer_id | идентификатор | BIGSERIAL | PRIMARY KEY |
| nickname | уникальный логин | VARCHAR(50) | UNIQUE, NOT NULL |
| email | электронная почта | VARCHAR(255) | UNIQUE, NOT NULL, CHECK (email ~* '^[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,4}$') |
| registered_at | дата и время регистрации | TIMESTAMPTZ | DEFAULT CURRENT_TIMESTAMP, NOT NULL |
| | | | |
| 2. Repository  |
| repository_id | идентификатор | BIGSERIAL | PRIMARY KEY |
| name | название репозитория | VARCHAR(100) | NOT NULL |
| url | ссылка на репозиторий | VARCHAR(1000) | UNIQUE, NOT NULL, CHECK (url ~* '^(https?://\|ssh://)?([a-z0-9_.-]+@)?[a-z0-9.-]+([:/][a-z0-9./~_?&=+-]+)?$') |
| | | | |
| 3. Repository Collaborator |
| developer_id | идентификатор разработчика | BIGINT | PRIMARY KEY, FOREIGN KEY (Developer) ON DELETE CASCADE |
| repository_id | идентификатор репозитория | BIGINT | PRIMARY KEY, FOREIGN KEY (Repository) ON DELETE CASCADE |
| access_role | уровень доступа | VARCHAR(10) | DEFAULT 'READER', NOT NULL, CHECK (access_role IN ('OWNER', 'MAINTAINER', 'DEVELOPER', 'READER')) |
| added_at | дата добавления в проект | TIMESTAMPTZ | DEFAULT CURRENT_TIMESTAMP, NOT NULL |
| | | | |
| 4. Branch |
| branch_id | идентификатор | BIGSERIAL | PRIMARY KEY |
| repository_id | к какому репозиторию относится | BIGINT | FOREIGN KEY (Repository) ON DELETE CASCADE, NOT NULL |
| branch_name | название ветки | VARCHAR(100) | NOT NULL |
| is_protected | защищена ли ветка | BOOLEAN | DEFAULT FALSE, NOT NULL |
| *constraint* | уникальность имени ветки в репозитории | | UNIQUE (repository_id, branch_name) |
| | | | |
| 5. Pipeline |
| pipeline_id | идентификатор пайплайна | BIGSERIAL | PRIMARY KEY |
| branch_id | идентификатор ветки | BIGINT | FOREIGN KEY (Branch) ON DELETE CASCADE, NOT NULL |
| trigger_dev_id | кто из разработчиков запустил пуш | BIGINT | FOREIGN KEY (Developer) ON DELETE SET NULL |
| commit_hash | хэш коммита | VARCHAR(128) | NOT NULL |
| status | текущий статус сборки | VARCHAR(10) | DEFAULT 'PENDING', NOT NULL, CHECK (status IN ('PENDING', 'RUNNING', 'SUCCESS', 'FAILED', 'CANCELLED')) |
| created_at | время создания | TIMESTAMPTZ | DEFAULT CURRENT_TIMESTAMP, NOT NULL |
| started_at | время начала выполнения | TIMESTAMPTZ | CHECK (started_at IS NULL OR started_at >= created_at) |
| finished_at | время окончания | TIMESTAMPTZ | CHECK (finished_at IS NULL OR started_at IS NULL OR finished_at >= started_at) |
| | | | |
| 6. Runner |
| runner_sk | ключ версии конфигурации| BIGSERIAL | PRIMARY KEY |
| runner_uuid | идентификатор сервера | UUID | NOT NULL, DEFAULT gen_random_uuid () |
| os_type | операционная система | VARCHAR(50) | NOT NULL |
| ram_mb | объем оперативной памяти в МБ | INT | > 0, NOT NULL |
| status | состояние раннера | VARCHAR(7) | DEFAULT 'IDLE', NOT NULL, CHECK (status IN ('IDLE', 'BUSY', 'OFFLINE')) |
| valid_from | начало действия версии конфигурации | TIMESTAMPTZ | DEFAULT CURRENT_TIMESTAMP, NOT NULL |
| valid_to | окончание действия версии | TIMESTAMPTZ | DEFAULT '9999-12-31 23:59:59', NOT NULL |
| is_current | флаг актуальности текущей версии | BOOLEAN | DEFAULT TRUE, NOT NULL |
| *constraint* | логика временных интервалов | | CHECK (valid_to >= valid_from) |
| | | | |
| 7. Job |
| job_id | идентификатор | BIGSERIAL | PRIMARY KEY |
| pipeline_id | к какому пайплайну относится | BIGINT | FOREIGN KEY (Pipeline) ON DELETE CASCADE, NOT NULL |
| runner_sk | на какой версии раннера выполняется | BIGINT | FOREIGN KEY (Runner) ON DELETE SET NULL |
| job_name | название шага | VARCHAR(100) | NOT NULL |
| required_os | требуемая ОС для выполнения | VARCHAR(50) | NOT NULL |
| status | текущий статус задачи | VARCHAR(10) | DEFAULT 'PENDING', NOT NULL, CHECK (status IN ('PENDING', 'RUNNING', 'SUCCESS', 'FAILED', 'CANCELLED')) |
| started_at | время начала выполнения задачи | TIMESTAMPTZ | |
| finished_at | время окончания выполнения задачи | TIMESTAMPTZ | CHECK (finished_at IS NULL OR started_at IS NULL OR finished_at >= started_at) |
| | | | |
| 8. Artifact |
| artifact_id | идентификатор | BIGSERIAL | PRIMARY KEY |
| job_id | в результате какой задачи создан | BIGINT | FOREIGN KEY (Job), NOT NULL |
| file_name | имя файла | VARCHAR(255) | NOT NULL, CHECK (file_name ~ '^[^\s/\\][^\/\\]{0,254}$') |
| file_path | путь в хранилище | VARCHAR(512) | NOT NULL, CHECK (file_path ~ '^(/[^/]+)+$') |
| size_bytes | размер файла в байтах | BIGINT | >= 0, NOT NULL |
| uploaded_at | дата и время загрузки | TIMESTAMPTZ | DEFAULT CURRENT_TIMESTAMP, NOT NULL |
