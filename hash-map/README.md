# Реализация Hash Map на C++

Собственная шаблонная реализация хеш-таблицы с нуля. Максимально приближена к реализации из stl (g++).

##  Ключевые особенности

*   Поддержка произвольных типов ключей и значений.
*   Открытая адресация.
*   Автоматическое увеличение емкости таблицы при превышении `load_factor` для сохранения амортизированной сложности **O(1)**.
*   Реализованы стандартные методы (`insert`, `find`, `erase`) и поддержка итераторов.

## Технические детали

*   **Хеш-функция:** Является шаблонным параметром.
*   **Load Factor:** Таблица перестраивается, когда коэффициент заполнения превышает.

## Пример использования

```cpp
#include "unordered_map.h"
#include <iostream>
#include <string>

int main() {
    UnorderedMap<std::string, int> map;

    map.insert("CPU", 100);
    map.insert("RAM", 32);

    if (auto it = map.find("CPU"); it != map.end()) {
        std::cout << "Value: " << it->second << std::endl;
    }

    return 0;
}
```
