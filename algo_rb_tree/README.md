### Упорядоченное дерево

**Полное решение**

| Ограничения | Все языки (make) | Java 21 (Temurin JDK) |
| :--- | :--- | :--- |
| **Ограничение времени** | 2 секунды | 6 секунд |
| **Ограничение памяти** | 256Mb | 512Mb |
| **Ввод** | стандартный ввод или input.txt |
| **Вывод** | стандартный вывод или output.txt |

Реализуйте splay-дерево. Для получения **5 баллов** за задачу можно вместо этого реализовать B-дерево или красно-чёрное дерево.

Пусть S — множество чисел. Изначально S пусто. Обработайте запросы следующих видов:

*   `insert x`: вставить x в S. Если $x \in S$ до запроса, то запрос игнорируется.
*   `delete x`: удалить x из S. Если $x \notin S$ до запроса, то запрос игнорируется.
*   `exists x`: проверить, содержится ли x в S. Если $x \notin S$, выведите «true», иначе выведите «false».
*   `next x`: найдите в S наименьшее число, строго большее x. Если такого числа нет, выведите «none», иначе выведите его значение.
*   `prev x`: найдите в S наибольшее число, строго меньшее x. Если такого числа нет, выведите «none», иначе выведите его значение.
*   `kth k`: найдите k-ю порядковую статистику S в 0-индексации. Если $k \notin [0, |S|)$, выведите «none», иначе выведите значение *k*-й порядковой статистики.

#### Формат ввода

Входной файл состоит из нескольких строк, задающих запросы в приведённом выше формате.

Все числа во входном файле по абсолютному значению не превышают $10^9$.

#### Формат вывода

Выведите ответы на все запросы.

#### Пример

**Ввод**
```
insert 2
insert 5
insert 3
exists 2
exists 4
next 4
prev 4
delete 5
next 4
prev 4
kth 0
kth 2
```

**Вывод**
```
true
false
5
3
none
3
2
none
```
